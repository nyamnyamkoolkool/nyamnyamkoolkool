# UC-3. 가족 초대 사진 갤러리 (달력형)

> 아기 사진을 날짜별로 정리하고, 가족 구성원을 초대하여 함께 보고 추억을 쌓는다.

상위: [메인 시나리오](../main-scenario.md)

---

## UC-3.1 사진 업로드 및 일자별 정리

| 항목 | 내용 |
|---|---|
| Actor | 보호자, 초대된 가족 구성원 |
| 목표 | 아기 사진을 날짜별 달력에 정리하여 가족과 공유 |
| 사전조건 | 사진/카메라 권한, 가족 그룹 생성 또는 참여 |

**주 시나리오**
1. **[갤러리]** 탭 진입 → 달력 뷰가 기본 표시
2. 사진이 있는 날짜는 썸네일 미리보기 표시
3. 특정 날짜 탭 → 해당일 사진 리스트
4. **[+ 업로드]** 로 사진/영상 추가
   - 단일/다중 선택 지원
   - **자동으로 촬영일(EXIF) 기준 분류**
5. 사진별 캡션, 마일스톤 태그(첫 미소, 뒤집기 등) 부여
6. 업로드 시 가족 그룹 멤버에게 푸시 알림

**대안 흐름**
- **A1. 촬영일 정보 없음**: 업로드 시점 또는 사용자 지정 날짜로 분류
- **A2. 중복 사진**: 해시 비교 후 중복 알림 표시

## UC-3.2 가족 초대 및 권한 관리

| 항목 | 내용 |
|---|---|
| Actor | 보호자 (그룹 관리자) |
| 목표 | 가족 구성원을 그룹에 초대하여 사진/기록 공유 |

**주 시나리오**
1. **[가족]** 메뉴 → **[+ 초대]**
2. 초대 방법: 카카오톡 / SMS / 이메일 / 링크 복사
3. 초대 링크: 유효기간(7일) + 권한 레벨 부여
4. 초대받은 사람이 앱 설치 + 링크 클릭 → 가족 그룹 자동 합류
5. 멤버 리스트에서 권한 변경/제거 가능

**권한 레벨 (4단계, 2026-05-21 갱신)**
| 권한 | 가능한 동작 | 적합한 사용자 |
|---|---|---|
| **관리자 (ADMIN)** | 모든 기록 작성/수정/삭제, 멤버 관리, 권한 변경, AI 프로바이더 설정, 결제·구독 | 그룹 생성자 (보통 엄마/아빠) |
| **보호자 (GUARDIAN)** | 모든 기록 작성/수정, 본인 작성분만 삭제, 마일스톤 태깅, 의료 정보 접근 | 다른 부모 |
| **공동양육자 (CONTRIBUTOR)** | 일지·사진·울음분석 작성, **본인 작성분 수정 자유**, 삭제 불가, 의료 정보 접근 불가 | 시터, 이모/삼촌, 임시 양육자 |
| **뷰어 (VIEWER)** | 조회 + 본인 사진 업로드·삭제, 울음 분석 실행 | 양가 조부모 (기본 추천) |

**친족 관계 표시 (BabyGuardian.relation)** — 양가 그룹은 분리하지 않되, 명칭으로 자연스럽게 구분:
- 엄마 · 아빠
- 외할머니 · 외할아버지 (외가)
- 할머니 · 할아버지 (친가)
- 이모 · 외삼촌 · 고모 · 삼촌
- 시터 · 새부모 · 형제·자매 · 기타

신규 멤버 가입 시 자동 추천: `relation`을 선택하면 nickname이 자동 입력됨 ("외할머니" → 가족 알림에 "외할머니가 사진을 추가했어요" 형태로 표시).

**송신 측 알림 가시성 (notifyOthersAboutMyActivity)** — 멤버 본인이 자신의 활동을 다른 가족에게 알릴지 토글:
- 기본 true (가족 공유의 가치 보존)
- 시터가 활동 종료 후 일지만 입력하고 부모만 받게 하려면 false 권장 (조부모 알림 부담 감소)
- 수신측 NotificationPreference와 AND 조건 (둘 다 켜져 있어야 푸시 발송)

**보안**
- 초대 링크는 만료 + 단일 사용 토큰
- 가족 그룹 외부에는 사진/기록 접근 차단 (E2E 암호화)

## UC-3.3 달력 뷰 탐색 및 추억 되돌아보기

**주 시나리오**
1. 갤러리 달력에서 월 단위 이동 (스와이프/연도 점프)
2. **[추억]** 카드: "1년 전 오늘" 사진 자동 추천
3. **[성장 모아보기]**: 마일스톤 태그로 묶인 사진 모음 자동 생성
4. 연/월 단위 자동 앨범 생성, 가족과 공유

**부가 기능**
- 마일스톤별 모음(첫 미소, 첫 뒤집기, 첫 걸음 등)
- 슬라이드쇼 / BGM 첨부 영상 자동 생성
- 사진 검색: 날짜 범위 + 마일스톤 태그 필터
