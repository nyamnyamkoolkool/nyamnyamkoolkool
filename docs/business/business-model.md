## 5.5 앱인토스 채널 깔때기 전략 (2026-05-21 결정)

토스 미니앱은 **체험 채널**, 네이티브 앱은 **풀 기능 채널**로 명확히 분리.

### 5.5.1 토스 미니앱에서 제공
- ✅ 육아일지 (수유·수면·배변)
- ✅ AI 울음 분석 (UC-1.1, UC-1.2)
- ✅ 아기 프로필·성장·예방접종
- ✅ AI 맞춤 팁
- ✅ Pro 구독 결제 (Toss IAP)

### 5.5.2 토스 미니앱에서 제외 (네이티브 유도)
- ❌ 사진 갤러리 (UC-3.1, UC-3.3) → 사진 보기 시도 시 "**네이티브 앱에서 가족과 함께**" CTA
- ❌ 음성비서 (UC-2.5) → "더 빠른 입력은 네이티브 앱에서"
- ❌ Gemma 4 오프라인 분석 → "오프라인에서도 분석하려면 네이티브 앱"
- ❌ BYOK / 영상 백업

### 5.5.3 깔때기 (Funnel) KPI
| 단계 | 가설 전환률 |
|---|---|
| 토스 미니앱 진입 (토스 사용자 viral) | — |
| → 일지 1건 작성 | 60% |
| → 울음 분석 1회 사용 | 30% |
| → **"사진 보기" CTA 노출** | 50% (가족 초대 받은 사용자 비중) |
| → **네이티브 앱 설치** | **15~25%** |
| → 30일 잔존 | 50% |

### 5.5.4 채널별 사용자 비중·매출 영향
- 토스 사용자 비중 가정: 30% (앱인토스 초기 진입)
- 토스 → 네이티브 전환 후 비중: 약 5~7% (잔존)
- **토스 광고 매출은 네이티브의 30% 수준** (사진 갤러리 슬롯 부재)
- 단 **토스가 가입 채널 1순위** (가족 초대 viral과 결합) → 장기 LTV 기여 큼

## 6. 마케팅·획득 비용 (CAC)

### 6.1 채널별 추정
| 채널 | CAC | LTV (24개월 잔존, 광고+Pro 5%) | LTV/CAC |
|---|---|---|---|
| **앱인토스 viral** | ~₩2K | ~₩100K | **50×** |
| **가족 초대 viral** (Kakao 알림톡 활용) | ~₩500 | ~₩100K | **200×** |
| 구글 검색 (Pro 부모) | ~₩15K | ~₩100K | 6.7× |
| 인스타·페이스북 광고 | ~₩25K | ~₩100K | 4× |
| **카카오톡 알림톡 + 채널 광고** (Kakao Cloud 시너지) | ~₩15K | ~₩100K | 6.7× |

**카카오 인프라 시너지 (2026-05-21 Kakao Cloud 채택 반영)**:
- Kakao Cloud 채택으로 카카오톡 비즈니스 API·알림톡 통합·심사 우위
- AuthIdentity provider에 KAKAO_OAUTH 이미 포함 → 가입 1탭
- 가족 초대 발송 채널: **카카오톡 알림톡** 우선 → SMS 대비 60% 비용 절감 + 도달률 ↑
- 사진 본체가 Kakao Cloud에 있어 향후 카카오톡 공유·내보내기 시 추가 인프라 비용 ↓

LTV 가산 사유: 무료 사용자도 광고 수익으로 +₩2,800/월 기여 → 24개월 LTV ₩67K (무료) + ₩33K (Pro 가중) = ₩100K

### 6.2 viral coefficient
- 가족 1명 초대 → 평균 8명 전원 가입 시도 → 70% 수락률 → k-factor 5.6 (보수적 3)
- 6개월 내 사용자 8배 증가 가능 (이론치)
