## 2. 직접 경쟁자 — 쑥쑥찰칵 (Babyfolio, 제제미미)

> 사용자 본인이 현재 사용 중인 앱. **한국 시장에서 가장 직접적 경쟁자**.
> 출처: [App Store](https://apps.apple.com/kr/app/%EC%91%A5%EC%91%A5%EC%B0%B0%EC%B9%B5/id1509183009) · [공식 사이트](https://babyfolio.family/kr) · [플래텀 기사](https://platum.kr/archives/269327)

### 2.1 핵심 기능 (검증된 USP)
- **양가 가족 분리** (외가/친가 별도 관리) — 한국 특유 가족 갈등 방지. 매우 한국화된 UX
- **자동 사진 정리** (날짜·월별 타임라인)
- **매월 성장 영상 무료 자동 생성**
- **큰 글씨 모드** (조부모 명시 타겟)
- **웹 버전 출시** — 앱 미설치 조부모도 사진 보기 가능
- 양가 부모·친척·이모·삼촌까지 초대 가능

### 2.2 검증된 사용자 패턴 (사용자 본인 경험 + 공식 인터뷰)
> "한국 정책 연구원 조동철 원장님 아침 루틴 중 하나가 쑥쑥찰칵으로 손주 사진 보는 것" — 운영사 인터뷰
> "대부분 어르신들이 하루에도 몇 번이나 들어와서 손주 사진을 보고, 친구 만나면 자랑한다"

→ **조부모 진입 빈도 매우 높음** (가정 7회/일 → 10~15회/일 상향 검토)
→ 광고 노출 기회 우리 모델 가정보다 큼

### 2.3 수익 모델
- **광고 + 구독** (FamilyAlbum과 달리 광고 포함 모델 — 한국 시장 검증)
- 무료: 광고 시청, 일부 가족 공유 제한
- 유료: 광고 제거 + 무제한 저장 + 가족 공유 풀

### 2.5 학습할 점 (쑥쑥찰칵 → 우리)
1. **양가 가족 분리는 한국 시장 차별 핵심** — v1 데이터 모델에 `FamilyGroup.lineage_type`(`MATERNAL`/`PATERNAL`/`UNIFIED`) 추가 검토
2. **매월 성장 영상 자동 생성** — FamilyAlbum·쑥쑥찰칵 공통, 우리도 v1.x로 끌어올 가치
3. **웹 버전** — 토스 미니앱이 일부 대체하지만 일반 웹 브라우저 진입도 검토
4. **큰 글씨 모드 우선순위 상향** — 조부모 진입율 높은 시장이라 v1 필수
5. **광고 + 구독 한국 시장 검증됨** — 우리 모델 정합성 ↑

### 2.6 우리가 차별화할 부분 (사용자 경험 검증 기반 재정립)

**핵심 메시지**: "기록은 끝나고 입력해도 충분" — 일지 기능 자체가 아니라 **입력 UX**가 차별점.

1. **일지 입력 UX 혁신** (1순위 차별)
   - 쑥쑥찰칵 사용자 본인 검증: *"입력이 좀 불편"*
   - 우리 해결책: 사후 시각 입력 + 음성비서 ("아기 12시 30분에 잤어" → 자동 기록) + 직전 패턴 원터치 복제
   - 메모리 [feedback-nyamnyam-ux](#)의 액티브 인터랙션 최소화 원칙이 사용자 경험으로 검증됨

2. **울음 AI 분석** (2순위 차별)
   - 0~24개월 양육의 핵심 페인 포인트(울음 원인 미상)
   - 쑥쑥찰칵·FamilyAlbum 모두 없음

3. **AI 맞춤 팁 + 멀티 LLM·BYOK** — 누적 일지 데이터를 활용한 인사이트 (UC-4.3, UC-1.2)

4. **앱인토스 진입** — 쑥쑥찰칵 미진입 채널

## 3. Tinybeans의 교훈 — 광고 의존 위험

- 2014~2023: 광고 85% / 구독 15% 모델
- 2024년 광고 매출 폭락 → **구독료 87% 인상**
- 결과: 기존 구독자의 **92% 잔존** → 가족 사진 앱은 **lock-in 매우 강함**

**시사점 (우리 적용)**:
1. 광고 매출은 글로벌 트렌드상 감소 추세. 의존도 60% 이하 권장
2. 구독 가격은 인상 여지 큼 (92% 잔존 = 강한 lock-in)
3. **이메일 다이제스트**(조부모용 앱 없이 사진 받기)는 viral·CAC 절감의 핵심 채널
4. 광고 매출 의존도가 손익 분기 전제. 광고 매출 -50% 시 손익 분기 DAU 2배 증가

## 4. Huckleberry의 교훈 — AI 가격 책정 여지

- Plus $9.99 / Premium $14.99 = ₩13,000~20,000/월
- 핵심: **AI 수면 예측 알고리즘** (우리 UC-1.2 패턴 인사이트와 유사)
- 미국 Premium 부모는 AI 가치에 월 $15 지불 의향

**시사점 (우리 적용)**:
1. 우리 Pro ₩4,900은 **AI 가치 대비 매우 저렴**
2. 향후 Pro Plus 티어 (₩7,900~9,900) 도입 가능 — 추가 LLM 사용량·고급 분석
3. **한국 시장은 미국 대비 50~70% 가격 수준** → ₩7,900이 상한 추정

## 5. 조부모 진입 패턴 인사이트

### 5.1 관측된 패턴 (FamilyAlbum·Tinybeans 사례)
- 조부모는 **일평균 진입율 70~85%** (앱 또는 이메일)
- 진입 동기 90%+ **사진 보기**, 10% 알림 받기
- 평균 세션 길이: 3~5분 (사진 슬라이드 + 코멘트)
- 일지·분석에는 관심 낮음
- 평일 낮 + 저녁 8~10시 진입 집중

### 5.2 우리 모델 영향
- **광고 노출 예상치 재산정**: 조부모 진입율이 높을 경우 가족당 일 광고 노출 7회 → **10~12회** 가능
- eCPM $3 가정 시 가족당 월 광고 수익: $2.52 → **$3.5~4.2** 상향
- 가족당 마진 (Free): +₩2,800 → **+₩4,000~5,500** 상향 가능

### 5.3 권장 보강
| 항목 | 권장 | 사유 |
|---|---|---|
| **이메일 다이제스트** | 도입 검토 (v1.x) | 조부모 앱 미설치 vs 진입 마찰. Tinybeans·FamilyAlbum 공통 채택 |
| **알림톡 다이제스트** | 도입 (한국 특화) | 카카오톡 + 토스 채널 알림으로 조부모 진입 |
| **광고 노출 위치 — 사진 갤러리 사이** | **검토 중지** | 사진 갤러리는 조부모 핵심 화면. UX 손상 위험 |

| **포토북·상품 수익** | **v1.x 검토** | FamilyAlbum의 가장 큰 매출원. ARPU 상향 |

## 6. 우리 모델 보정 권장 (시장 인사이트 반영)

### 6.1 무료 가족 멤버 수
| 현재 | 시장 표준 | 권장 |
|---|---|---|
| **8명** | FamilyAlbum/Tinybeans **무제한** | **무제한으로 변경 검토** |

**Trade-off**:
- 가족 멤버 8 → 무제한 변경 시 가족당 활성 4 → 5~6명 증가 가능 → LLM 비용 25% 증가
- 무료 사용자당 광고 노출 7회 → 10~12회 증가 → 광고 수익 40% 증가
- **순효과: 광고 수익 증가 > LLM 비용 증가**
- 단 추정치이므로 출시 후 검증 필요

### 6.2 무료 사진 한도
| 현재 | 시장 표준 | 권장 |
|---|---|---|
| **10GB** | FamilyAlbum **무제한** | **사진 무제한, 영상만 Pro 한정** |

**사유**: 사진은 텍스트보다 저렴(Cloudflare R2 + 압축으로 ~$0.005/GB). 무제한 정책이 viral 핵심 동인. 영상은 Storage 부담 크므로 Pro 한정 유지.

### 6.3 광고 노출 위치 재검토
| 위치 | 이전 안 | 권장 |
|---|---|---|
| 홈 화면 하단 배너 | ✅ | ✅ 유지 |
| AI 팁 카드 사이 네이티브 | ✅ | ✅ 유지 |
| 갤러리 월별 전환 | ✅ | **❌ 제거** (조부모 핵심 화면 손상) |
| **갤러리 진입 시 인터스티셜** | — | **신규 검토** (30회당 1회) |
| 일/주간 요약 끝 | ✅ | ✅ 유지 |

### 6.4 추가 수익원 검토 (v1.x)
| 수익원 | 시장 사례 | 우리 적용 |
|---|---|---|
| **포토북 인쇄** | FamilyAlbum 큰 매출원 | 4~6개월 포토북 자동 생성 + 가족 단위 우편 발송 |
| **앨범 캘린더·달력** | FamilyAlbum | 1년 정리 캘린더 자동 |
| **마일스톤 카드** | 일부 미국 앱 | "첫 미소" 카드 인쇄·우편 |
| **포토 굿즈** (액자·머그) | 다양 | Pro 이상 가족 한정 할인 |

→ 모두 별도 결제, Pro와 독립. ARPU 상향 가능성 큼.

### 6.5 이메일·알림톡 다이제스트
| 항목 | 권장 |
|---|---|
| **이메일 다이제스트 (조부모)** | v1.x 도입 — 앱 미설치 조부모용 |
| **카카오 알림톡** | 한국 viral 핵심 채널 — v1 도입 |
| **토스 푸시** | 앱인토스 사용자 — 정책 확인 후 v1 |

## 8. 손익 분기 재시뮬레이션 (시장 인사이트 반영)

### 8.1 가정 변경
- 가족 멤버: 8 → **무제한** (활성 평균 5명)
- 무료 사진: 10GB → 무제한
- 광고 노출/가족: 7회 → **10회** (조부모 진입율 ↑)
- 광고 수익/가족 (활성 5명, 일 2회 노출): $0.63 → **$0.90/월**
- LLM 비용/가족: $0.27 → $0.34/월 (활성 5명)
- 인프라/가족: $0.13 → $0.20/월 (사진 무제한)

## 10. 위험·검증

| 항목 | 검증 방법 |
|---|---|
| 가족 무제한 → LLM 비용 폭증 위험 | Cloud Functions 가족당 일 한도 강제 (가족당 분석 20건/일 등) |
| 사진 무제한 → Storage 비용 증가 | Cloudflare R2 이전 시점 단축 (Early 단계에서 즉시) |
| 광고 거부감 vs 조부모 진입 빈도 | A/B 테스트 (광고 ON/OFF 코호트 30일 잔존) |
| FamilyAlbum 한국 점유율 | 정확한 MAU 통계 추가 조사 필요 (앱스토어 순위 + 데이터.ai) |
| 조부모 활성률 70~85% 가정 | 출시 30일 코호트 분석 |
| 포토북 수익 가능성 | FamilyAlbum 매출 공개 자료 + 단가 조사 (한국 인쇄 협력사) |
