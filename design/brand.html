잘 먹고, 잘 자는
다정한 동행.

냠냠쿨쿨의 마크는 둥근 얼굴에 감긴 미소 눈과 살짝 웃는 입꼬리 하나로 "방금 잘 먹은 만족"과 "포근하게 잠든 평온함"을 동시에 담습니다. 16px 파비콘부터 1024px 앱 아이콘까지 같은 표정으로 살아남도록 설계됐어요.

CONCEPT 디자인 컨셉 하나의 표정이 두 메시지를 전한다

"감긴 눈은 잠, 웃는 입은 만족"

위로 휘는 두 눈썹 모양의 감긴 눈은 동아시아 시각 문화에서 행복하게 잠든 표정으로 읽힙니다. 작은 입꼬리는 방금 잘 먹은 아기의 만족스러운 표정. 발그레한 두 볼은 건강함. 별도의 달이나 zZ, 우유병 같은 요소를 추가하지 않은 이유는 — 얼굴 하나로 양쪽 의미를 모두 담을 때 가장 절제되고, 가장 작은 크기에서도 살아남기 때문입니다.

USAGE 사용 가이드 하면 좋은 것 · 피해야 할 것

✓ Do

  • 코랄 메인 컬러 그대로 사용 — 채도·명도 임의 변경 금지
  • 마크 주변에 마크 높이의 25% 이상 여백(클리어 스페이스) 확보
  • 16px 이하에서는 favicon.svg 사용 (얼굴 디테일 단순화 버전)
  • 다크 배경에서는 icon-mono.svg를 흰색(currentColor=white)으로 사용
  • 워드마크 듀얼 컬러 유지 — 냠냠=코랄, 쿨쿨=인디고
  • 이중 의미 보존 — "잘 먹고 잘 잔다"가 핵심 메시지

✕ Don't

  • 얼굴에 zZ · 달 · 우유병 같은 추가 요소 덧붙이지 않기 (디자인 의도 훼손)
  • 그라데이션·그림자·테두리 추가하지 않기 — 플랫 유지
  • 마크를 회전·기울이기·찌그러뜨리지 않기
  • 워드마크와 마크 비율을 임의 조정하지 않기 (logo.svg 그대로 사용)
  • 코랄을 보라/핑크/오렌지로 임의 대체하지 않기
  • 아이콘 배경 라운드 비율 변경 금지 (iOS squircle 비율 기준)
